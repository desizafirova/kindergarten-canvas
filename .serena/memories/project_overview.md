# Project Overview

**Project Name:** kindergarten-canvas (Little Stars Kindergarten / ДГ №48 Ран Босилек)

## Description
A modern, playful kindergarten website built with React and TypeScript. The site showcases kindergarten programs, services, facility information, admission process, and contact capabilities.

## Technology Stack

### Core Framework
- **React** 18.3.1 - UI library
- **TypeScript** 5.8.3 - Type safety
- **Vite** 5.4.19 - Build tool and dev server
- **React Router DOM** 6.30.1 - Client-side routing

### UI & Styling
- **Tailwind CSS** 3.4.17 - Utility-first CSS framework
- **shadcn-ui** - Component library built on Radix UI primitives
- **Framer Motion** 12.29.0 - Animation library
- **Lucide React** - Icon library

### Forms & Data
- **React Hook Form** 7.61.1 - Form handling
- **Zod** 3.25.76 - Schema validation
- **@tanstack/react-query** 5.83.0 - Data fetching and caching

### Testing
- **Vitest** 3.2.4 - Unit testing framework
- **@testing-library/react** 16.0.0 - Component testing utilities

## Design System
- **Primary Color (Sunny Yellow):** hsl(45 95% 55%) - Main CTA buttons
- **Secondary Color (Sky Blue):** hsl(200 85% 60%) - Secondary buttons
- **Accent Color (Coral):** hsl(12 90% 65%) - Highlights
- **Fonts:** Fredoka (display), Nunito (body)

## Key Entry Points
- `index.html` - HTML entry point
- `src/main.tsx` - React app bootstrapping
- `src/App.tsx` - Main app with routing
- `src/pages/Index.tsx` - Home page
